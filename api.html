Amazon Scraper API Documentation

This API allows you to scrape product details, reviews, offers, and search results from Amazon.

Get Product Details

Retrieve details of a specific Amazon product.

            GET /products/:productId

Example:

                /products/B07G3QMPB5

Get Product Reviews

Retrieve reviews of a specific Amazon product.

            GET /products/:productId/reviews

Example:

                /products/B07G3QMPB5/reviews

Get Product Offers

Retrieve offers for a specific Amazon product.

            GET /products/:productId/offers

Example:

                /products/B07G3QMPB5/offers

Search for Products

Search for products on Amazon by a query.

            GET /search/:searchQuery

Example:

                /search/laptop

Usage

To use this API, make GET requests to the specified endpoints as shown in the examples above. The API will return JSON responses with the scraped data.

API Base URL

The base URL for the API is not specified in this documentation, but you can define it in your code using the API key provided.

Remember to handle errors properly in your application, as the API might return 500 Internal Server Error for invalid requests.
